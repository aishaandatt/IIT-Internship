# Face_Recognition
## Running
1. Installing dlib
```git clone https://github.com/davisking/dlib.git```
```cd dlib```
```mkdir build; cd build; cmake ..; cmake --build .```
```cd ..```
```python3 setup.py install```

2. ```brew install cmake```
3. ```pip install face_recognition```

>For Windows
